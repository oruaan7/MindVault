# Mind Vault

O Mind Vault é uma plataforma de produtividade pessoal projetada para centralizar informações importantes em um só lugar.

## Módulos

- Autenticação
- Painel
- Hábitos
- Finanças
- Metas
- Notas
- Projetos
- Estudos
- Lembretes
- Gamificação

## Tecnologias

### Backend
- Java 21
- Spring Boot
- Spring Security
- JPA / Hibernate
- PostgreSQL

### Frontend
- Angular
- TypeScript
- SCSS

### DevOps
- Docker
- Docker Compose
- GitHub Actions
