# Domain Dictionary

This document defines the ubiquitous language used throughout the MindVault project.

| Term | Definition |
|------|------------|
| User | An authenticated person who uses the platform. |
| Habit | A recurring activity defined by the user. |
| Habit Record | A historical record indicating the completion of a habit. |
| Goal | A measurable objective defined by the user. |
| Project | A collection of tasks or activities with a specific objective. |
| Note | A free-form text created by the user. |
| Study Session | A record of a study activity performed by the user. |
| Financial Transaction | A financial operation representing either income or expense. |
| Category | A classification used to organize financial transactions. |
| Reminder | A scheduled notification or event created by the user. |
| XP Transaction | A record representing experience points earned or lost. |
| Dashboard | A consolidated view of the user's productivity metrics. |
